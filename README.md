# School Money Management System

A console-based Java application to manage school finances including student fee collection and teacher salary payments.

## Features
- Fixed student fee structure
- Partial fee payment support
- Salary payment with sufficient fund validation
- Real-time school balance tracking

## Tech Stack
- Java
- OOP
- Collections
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run Main.java
